Annual Event

Title: Mozilla Omaha's HTML Application Talk

Description:
Mozilla Omaha's HTML Talk is a forum chat that discusses current HTML Application Development projects. We will try to cover applications that have a real impact on how we use the social aspect of the web. What we are looking to gain in these talks is a sense of inclusion and the spread of ideas. Our plan is to network with other Mozilla Clubs so that we can build towards inclusion and start spreading ideas.
